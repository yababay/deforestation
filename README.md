# Скорость обезлесенья в связи с развитием металлургии

В Средние века человечество потребляло древисину в очень больших объемах. По сути это был единственный доступный материал и для строительства, и для кораблестроения, но особенно много деревьев стали вырубать после начала бурного развития черной металлургии. Целые страны лишались лесов за считанные столетия, если не за десятелетия. Какова была скорость обезлесенья?
